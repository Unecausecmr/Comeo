Как я смогу получить деньги, которые соберет мой проект?

Деньги можно будет перевести на банковский счет. Для этого нужно будет указать реквизиты в настройках.

Если у меня еще нет банковского счета, смогу ли я создать проект и собирать деньги?

Если проект собирает сумму меньше 5000 лей, тогда автор проекта может получить деньги наличными.

Какой % от собранных денег остается у платформы?

Comeo это некоммерческая организация. Когда автор проекта получает деньги, у Comeo остается X % необходимых для оплаты комиссий денежных переводов и технической поддержки сайта (см. также Цели Comeo)

Если мой проект соберет только часть запланированной суммы, смогу ли я получить деньги?

Да, часть денег возможно будет получить, если при создании проекта выбрать фиксированный тип сбора денег.
Если выбран гибкий тип, тогда деньги вернутся вкладчикам.

Возможно ли предоставлять вознаграждения для тех кто поддерживает мою идею или продукт?

При создании проекта можно будет указать, планируете ли вы предоставлять вознаграждение вкладчикам.
Возможно будет определить разные вознаграждения для конкретных сумм.

Как можно будет передать деньги автору идеи? Какие способы оплаты поддерживает сайт?

Сайт поддерживает оплату банковскими картами Visa/Mastercard. Также возможна оплата без банковской карты, наличными, через платежный терминал.
Инфо обо всех денежных переводах сохраняется в системе Comeo и в итоге информация обо всех вкладах видна на странице конкретного проекта.
